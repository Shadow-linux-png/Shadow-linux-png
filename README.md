## 🏗️ Featured Engineering Systems

### 📍 [IndiRoadsSim — Road Digital Twin](https://github.com/Shadow-linux-png/IndiRoadsSim)
**The Problem:** Standard simulators lack the chaotic geometric complexity of Indian road networks.  
**The Solution:** A high-fidelity Digital Twin leveraging OSM data and custom geometric modeling to simulate real-world traffic patterns for autonomous vehicle training.
*   **Tech Stack:** Python, OSMnx, Shapely, SUMO.

### 🚦 [Traffic Pothole Simulator](https://github.com/Shadow-linux-png/Traffic_pothole_simulator)
**The Problem:** Anomaly detection usually happens in isolation from traffic flow.  
**The Solution:** An integrated pipeline where YOLOv8 detects anomalies (potholes) and triggers real-time rerouting logic within a SUMO environment to optimize city-wide throughput.
*   **Tech Stack:** YOLOv8, OpenCV, TraCI, Python.

### 🎭 [EmotionAutoCity](https://github.com/Shadow-linux-png/EmotionAutoCity)
**The Problem:** Human-City interaction is static.  
**The Solution:** A speech-emotion recognition engine that drives adaptive response logic in urban environments.
*   **Tech Stack:** Librosa, Scikit-learn, FastAPI.
