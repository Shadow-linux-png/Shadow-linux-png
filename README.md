## 🚀 Featured Project

### 🌍 [ClimateCrowd-TrafficTwin](https://github.com/Shadow-linux-png/ClimateCrowd-TrafficTwin.git)  
*A Climate + Crowd Resilient Smart Traffic Digital Twin*  

- 🛣️ Simulates **urban traffic junctions** with vehicles + pedestrians  
- 🌧️ Models **weather impacts** (rain, heat) on flow & breakdowns  
- 👥 Handles **crowd surges** (festivals, stadium exits)
